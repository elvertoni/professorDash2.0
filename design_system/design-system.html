01

No-Line Rule

Bordas sólidas 1px são proibidas para seccionar. Limites vêm de shifts de superfície: uma camada low contra a base cria uma "soft edge" mais sofisticada que um traço duro.

soft edges only
02

Tonal Layering

Profundidade sem sombra web 2.0. Camadas obsidian empilhadas: low → surface → high → highest. Um card high sobre a base "levita"; um low "recede".

depth by density
03

Glass & Gradient

Vidro para navegação flutuante e overlays (70% opacidade + blur 24px). Gradiente 45° no CTA primário dá "alma" — evita o visual flat-and-dead.

soul, not flat
04

Intentional Asymmetry

Romper a grade 12-col rígida. Blocos de texto offset da imagem, densidades de fundo alternadas. Tensão visual que guia o olhar aos pontos de conversão.

break the grid
05

Negative Space

Espaço em branco é o divisor. Se parece vazio, adicione mais espaço — não mais elementos. O aluno respira entre conceitos.

space is content
★

Tipografia autoritativa

Geist com tracking apertado (−0.02em) nos títulos. Labels em maiúsculas com 0.05em. Contraste de escala = tensão editorial de alta moda.

contrast is king

A sintaxe do design editorial no século XXI

Uma exploração profunda sobre como tipografia e espaço negativo definem a autoridade digital em plataformas de conteúdo premium.

01. O paradigma da legibilidade

Diferente dos layouts tradicionais, o design editorial foca no ritmo vertical. A transição entre o que é lido e o que é visualizado deve ser fluida, quase invisível. O uso do Geist permite precisão técnica mantendo o calor humano da tipografia clássica.

:::conceito

Espaço negativo não é vazio — é a pausa que dá sentido ao conteúdo. O silêncio entre as notas define a música.

"O design não é apenas o que parece e o que se sente. Design é como funciona." — Steve Jobs, reinterpretado para a era do Atelier Digital.

Grades assimétricas

Romper a simetria cria tensão visual e guia o olhar do estudante para os pontos de conversão mais importantes.

Micro-interações

Pequenos movimentos que validam a ação do usuário, transformando uma aula passiva em uma experiência tátil.

Estrutura de código exemplar

function applyEditorial() {
  const spacing = "32px";
  return { layout: "asymmetric", depth: "tonal" };
}
:::atencao

Nunca use sombras pretas puras em cards — elas parecem lodosas. Use camadas de superfície ou sombras tinted (com a cor da marca).

02. Exercício prático

  1. 1Identifique três áreas da interface onde o espaço negativo poderia ser expandido para melhorar a leitura.
  2. 2Substitua bordas sólidas por mudanças tonais entre surface-low e surface-high.
  3. 3Implemente o modo de apresentação e teste o contraste a 3 metros de distância.
:::dica

Apresente para um colega e pergunte onde os olhos dele pousam primeiro. Se não for no ponto de conversão, ajuste a assimetria.
